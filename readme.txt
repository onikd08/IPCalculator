*** python version 3 is needed ***

How to run the code on linux machine or MAC:

1. copy or store the "sourcecode.py" file into a directory.
2. open terminal and go to the directory which contains the "sourcecode.py" file using cd.
3. Now type the command: python3 sourcecode.py IPADDRESS/PREFIX
Example: python3 sourcecode.py 192.168.0.15/24
and press enter.

The output will be desplayed in the terminal.

For the example mentioned above, the output will be like this:

sh-4.4$ python3 sourcecode.py 192.168.0.15/24
